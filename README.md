# Școala de Dans — Proiect Baze de Date (Oracle)

## Descriere Proiect
Proiectul reprezintă o **bază de date completă** pentru gestionarea unei școli de dans. Sistemul administrează **elevi, instructori, cursuri, stiluri de dans, abonamente, plăți, săli și feedback**. Include modelare conceptuală, normalizare și implementare SQL.

## Funcționalități
- Evidență elevi, instructori și stiluri de dans  
- Gestionare cursuri și săli, alocare instructor  
- Abonamente și plăți pentru cursuri  
- Colectare feedback de la cursanți  
- Interogări SQL complexe și optimizate (subcereri, funcții pe șiruri/date, CASE, OUTER JOIN, TOP‑N)  
- Exemple de **isolation levels** și planuri de execuție  
- Tabele normalizate până la 3NF  

## Tehnologii
- Oracle SQL Developer / Oracle Database  
- SQL (DDL, DML, interogări complexe)  
- Modelare conceptuală & ERD  

## Structura Proiectului
- Tabele: ELEV, INSTRUCTOR, STIL, CURS, ABONAMENT, BANI, SALA, FEEDBACK, PARTICIPA  
- Secvențe, scripturi SQL și diagrame ER & conceptuale
